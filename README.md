## English

### What is this?

Day1 Global Briefing is an open-source market intelligence dashboard that aggregates real-time data from multiple financial sources into a single, beautiful dark-themed interface. It covers US stocks, cryptocurrencies, market sentiment indices, BTC on-chain metrics, and AI-generated daily analysis.

### How it works

```
                    ┌─────────────────────────────────┐
                    │       Next.js Frontend (SWR)     │
                    │   Polls API routes every 5 min   │
                    └──────────────┬──────────────────┘
                                   │
                    ┌──────────────▼──────────────────┐
                    │     /api/market-data (aggregator)│
                    │     60s in-memory cache          │
                    └──────────────┬──────────────────┘
                                   │
          ┌────────────┬───────────┼───────────┬────────────┐
          ▼            ▼           ▼           ▼            ▼
    Yahoo Finance   OKX API   alternative.me  CoinGlass   Finnhub
    (Stocks/VIX/    (Crypto)  (Fear & Greed)  (BTC chain  (News)
     Gold/DXY)                                 metrics)
```

**Architecture overview:**

- **Frontend**: React (Next.js 14 App Router) + SWR for client-side data fetching with auto-refresh
- **Backend**: Next.js API routes aggregate data from 6+ sources concurrently, with in-memory caching
- **AI Analysis**: Daily market analysis generated by Claude AI, stored in Upstash Redis
- **Audio Briefing**: TTS audio generated via OpenAI, stored in Vercel Blob
- **Cron Job**: Vercel Cron or GitHub Actions triggers daily analysis generation + Telegram push

### Data Sources

2. **Sentiment** - Fear & Greed indices, VIX, market mood analysis

### Getting Started

#### Option 1: Visit the live site

Go to **[brief.day1global.xyz](https://brief.day1global.xyz/)** - no setup needed.

#### Option 2: Self-host

1. **Clone the repo**

```bash
git clone https://github.com/star23/day1global-briefing.git
cd day1global-briefing
```

2. **Install dependencies**

```bash
npm install
```

3. **Configure environment variables**

```bash
cp .env.example .env.local
```

Edit `.env.local` and fill in your API keys. See the table below for details:

| Variable | Description | Required |
|----------|-------------|----------|
| `CRON_SECRET` | Secret key for cron job authentication | Recommended |
| `ANTHROPIC_API_KEY` | Claude AI for daily market analysis | Yes |
| `FINNHUB_API_KEY` | US stock prices + news ([finnhub.io](https://finnhub.io/)) | Yes |
| `CMC_API_KEY` | CoinMarketCap fear/greed data | Optional |
| `COINGLASS_API_KEY` | BTC on-chain metrics ([coinglass.com](https://www.coinglass.com/pricing)) | For BTC tab |
| `OPENAI_API_KEY` | TTS audio generation | Optional |
| `BLOB_READ_WRITE_TOKEN` | Vercel Blob for audio storage | Optional |
| `KV_REST_API_URL` | Upstash Redis URL | Yes |
| `KV_REST_API_TOKEN` | Upstash Redis token | Yes |

4. **Run locally**

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000)

5. **Deploy to Vercel**

- Push to GitHub
- Import project in [Vercel](https://vercel.com)
- Add environment variables in Vercel dashboard
- Done!

### Tech Stack

- **Next.js 14** (App Router) + TypeScript
- **SWR** for client-side data fetching
- **Claude AI** (Anthropic) for market analysis
- **OpenAI** for TTS audio
- **Upstash Redis** for data caching
- **Vercel** for deployment + cron + blob storage
- **Geist Sans** + **Noto Sans SC** fonts

## 中文

### 这是什么？

Day1 Global Briefing 是一个开源的每日投资情报看板，实时聚合多个金融数据源，以深色科技感界面呈现美股、加密货币、市场情绪指数、BTC 链上指标及 AI 每日分析。

### 实现原理

```
                    ┌─────────────────────────────────┐
                    │     Next.js 前端 (SWR 轮询)       │
                    │     每 5 分钟自动刷新数据          │
                    └──────────────┬──────────────────┘
                                   │
                    ┌──────────────▼──────────────────┐
                    │    /api/market-data (数据聚合层)   │
                    │    60 秒内存缓存                   │
                    └──────────────┬──────────────────┘
                                   │
          ┌────────────┬───────────┼───────────┬────────────┐
          ▼            ▼           ▼           ▼            ▼
    Yahoo Finance   OKX API   alternative.me  CoinGlass   Finnhub
    (美股/VIX/       (加密货币)  (恐贪指数)     (BTC链上     (新闻)
     黄金/美元)                                 指标)
```

**架构概览：**

- **前端**：React (Next.js 14 App Router) + SWR 客户端数据获取，自动轮询刷新
- **后端**：Next.js API 路由并发聚合 6+ 数据源，内存缓存防止限流
- **AI 分析**：Claude AI 每日生成市场分析，存储在 Upstash Redis
- **语音早报**：OpenAI TTS 生成音频，存储在 Vercel Blob
- **定时任务**：Vercel Cron 或 GitHub Actions 触发每日分析生成 + Telegram 推送
